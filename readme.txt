Requirements:
    Python 3
    Run > pip install -r requirements.txt
    or
    Run > pip install numpy matplotlib

To Run the tests move into the test folder
Run > cd Tests

Run the XOR Test:
Run > python XOrTests.py

Run the Sin Test:
Run > python SinTests.py

Run the Letter Test:
Run > python LetterTests.py

This will rewrite the files that are already in the output folders if you wish you can delete the content that is already there.
